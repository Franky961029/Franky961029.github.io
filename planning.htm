Propiedades basicas de CSS, Display: inline, block, inline-block,

CSS: Es un lenguaje que se utiliza en documentos de HTML. Organizando la presentacion de una pagina web.

Display: Es una propiedad que permite controlar las estructuras. Los elementos poseen un valor por defecto sea block o inline.

Block: Hace que el elemento se convierta en un bloque

Inline: El elemento se rendizara en linea con otros elementos

Inline-block: El elemento mezcla entre las dos anteriores.
